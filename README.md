CSStudyRoom
แอปพลิเคชันเครือข่ายสังคมเพื่อการเรียนรู้สำหรับนักศึกษา CS KMUTNB

CSStudyRoom เป็นแพลตฟอร์มเว็บแอปพลิเคชันที่ออกแบบมาเพื่อสนับสนุนการเรียนรู้ร่วมกันของนักศึกษาภาควิชาวิทยาการคอมพิวเตอร์ โดยเป็นตัวกลางในการนัดหมายติววิชาต่างๆ และเป็นพื้นที่สำหรับถาม-ตอบปัญหาทางวิชาการ

ฟีเจอร์หลักของระบบ
ระบบนัดติว (Tutor Booking): ผู้ใช้สามารถสร้างโพสต์นัดติวโดยระบุวิชา สถานที่ วันเวลา และจำนวนที่รับสมัครได้

ระบบถาม-ตอบ (Q&A Community): พื้นที่สำหรับตั้งคำถามแยกตามหมวดหมู่วิชา พร้อมระบบกดถูกใจและแสดงความคิดเห็น

ระบบแชทกลุ่ม (Group Messaging): รองรับการสื่อสารแบบเรียลไทม์เพื่อพูดคุยรายละเอียดเพิ่มเติมในแต่ละวิชา

ปฏิทินกิจกรรม (Study Calendar): แสดงกำหนดการนัดติวทั้งหมดในรูปแบบปฏิทินเพื่อให้ง่ายต่อการติดตาม

เทคโนโลยีที่ใช้
Frontend: React 19, Vite, React Icons

Backend: Firebase Firestore (Database), Firebase Auth (Authentication)

Tools: ESLint สำหรับตรวจสอบคุณภาพโค้ด และ Browser Image Compression สำหรับจัดการขนาดรูปภาพ

โครงสร้างโปรเจกต์
โปรเจกต์นี้ได้รับการปรับปรุงโครงสร้างใหม่แบบ Feature-based เพื่อให้ง่ายต่อการบำรุงรักษาและขยายระบบในอนาคต:

src/api/: ส่วนจัดการ Logic การติดต่อฐานข้อมูล (Service Layer) และ Model ข้อมูลกลาง

src/features/: ส่วนประกอบหลักของแอปพลิเคชันแยกตามฟีเจอร์ (Tutor, QA, Chat, Profile, Admin)

src/components/common/: ส่วนประกอบ UI ที่ใช้ซ้ำในหลายหน้าของระบบ

src/hooks/: Custom Hooks สำหรับจัดการ Logic และการดึงข้อมูล

การติดตั้งเพื่อพัฒนาต่อ
คัดลอกโปรเจกต์จาก GitHub:
git clone https://github.com/kaimairuh/csstudyroom.git

ติดตั้งแพ็คเกจที่จำเป็น:
npm install

ตั้งค่าไฟล์ .env.local โดยระบุค่า Config จาก Firebase Project ของคุณ

เริ่มรันระบบในโหมดพัฒนา:
npm run dev
